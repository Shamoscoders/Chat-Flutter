# ChatFlutter

Realtime Chat application using FireStore and build with flutter

![Screenshot](screenshot/01.jpeg)  ![Screenshot](screenshot/02.jpeg)  ![Screenshot](screenshot/03.jpeg)

## Getting Started

to run this project

```
  flutter pub get

  flutter run

```
